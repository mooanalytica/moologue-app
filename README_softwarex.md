MooLogue — ML-Driven Bioacoustic Analysis for Dairy Cattle Vocalizations
SoftwareX – Developer & Research Documentation
Overview

MooLogue is a cross-platform mobile application for analyzing dairy cow vocalizations using Flutter, Python-based machine learning, and a serverless backend built on AWS Lambda, S3, API Gateway, and DynamoDB.

The application performs real-time audio capture, cloud inference, Mel-spectrogram generation, CNN-based classification, and farmer-facing decision support.

This file serves as the developer and research-focused documentation for the SoftwareX submission (v1.0 scientific release).
